Requirement Sync Service v2.4

API 接口定义

POST /v2/ledger/sync

请求参数 (Request Parameters)

名称 (Name) 类型 (Type) 描述 (Description)
ledger_id String 目标账本的唯一标识符。必须是有效的 UUID 格式。
sync_mode Enum 同步模式。可选值：delta, full。
timestamp Integer Unix 时间戳（毫秒级）。用于检测版本冲突。

响应代码 (Response Codes)

200

OK

数据同步成功启动。

400

Bad Request

请求格式错误或参数校验失败。

401

Unauthorized

API 密钥缺失或无效。

409

Conflict

版本冲突，请合并后重试。

curl -X POST "https://api.obsidianflux.com/v2/ledger/sync" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d {
    "ledger_id": "flux_99x_01j",
    "sync_mode": "delta",
    "timestamp": 1715234890000
  }
Response Body 200 OK 
{
  "status": "success",
  "sync_id": "sync_v_7721",
  "records_processed": 128,
  "latency_ms": 42
}
