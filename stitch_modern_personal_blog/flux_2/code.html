v2.4.0 2024年11月

新特性

  • 引入 Flux Core v2 引擎，支持更复杂的异步数据流处理。
  • 新增 Obsidian 节点 可视化编辑器，实现无代码逻辑编排。
  • 支持原生的多云部署环境，适配 AWS 与 GCP 的弹性扩展。

功能优化

  • 核心运行时内存占用降低了 34%，响应速度提升 2x。
  • 改进了 TopNavBar 的响应式逻辑，提升在移动端的流畅度。
  • 优化了全局搜索算法，支持全文索引。

缺陷修复

  • 修复了在某些极端场景下 SideNavBar 状态同步不一致的问题。
  • 解决了深色模式下某些 CSS 变量渲染冲突的 Bug。
  • 修复了 API 文档中部分示例代码拼写错误。

v2.3.5 2024年10月

新特性

  • 上线全新的 Beta 控制台。
  • 增加对 WebAssembly 模块的实验性支持。

功能优化

  • 重构了基础组件库，减少了 12kb 的打包体积。
  • 提升了 WebSocket 在高并发下的稳定性。

缺陷修复

  • 修复了认证 Token 在过期前无法自动刷新的问题。
